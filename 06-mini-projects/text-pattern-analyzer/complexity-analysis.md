# Complexity Analysis

 Function                   Time Complexity      Space Complexity 
------------                  ------                   ------     
 isPalindrome()                  O(n)                   O(n)       
 findPalindromeBreaks()          O(n × m)              O(k)       
 findRepeatedPhrases()          O(n × p)               O(n)                   
 analyzeTexts()               O(t × n × p)             O(t)       

Where:

- n = number of words
- m = average word length
- p = phrase length
- t = number of texts
- k = number of non-palindromes
