# Find Most Expensive Product

## Description
Find the most expensive product from an array of product objects.

## Example 

Input:
[
    {name: "Phone", price: 500},
    {name: "Laptop", price: 1200}
]
Output:

{
    name: "Laptop",
    price: 1200
}
## Concepts Practiced

- Arrays
- Objects
- Loops
- Comparsion Logic
- Maximum Tracking Pattern

## Complexity
Time Complexity: O(n)
Space Complexity: O(1)
